# Agents Guide

## 沟通规范

- 使用 `中文` 进行交流。
- 使用 `英文` 进行代码编写和注释。

## 开发规范

- 开始前阅读 `docs/AI_PLUGIN_EXTENSION_GUIDE.md` 以了解插件开发的基本要求和流程。
- 单个代码文件不超过500行。
- 优先选择已有的依赖，避免重复发明轮子。
- 可以读取项目目录范围之外文件，但禁止对项目目录范围之外的任何文件进行修改。
- 任务开始前，先检查是否已进入Python虚拟化环境，如果没有，请先执行 `source .venv/bin/activate`，不要反复检查和执行，仅在任务开始前执行一次即可。
